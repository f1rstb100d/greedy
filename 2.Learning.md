# Forword Chaining Algorithm and Backward Chaining Algorithm
```
例：if A and C then F
if A and E then G
if B then E
if G then D
求证 if A and B then D
正向的话要依据条件AB按条件顺序多次循环，看新得到什么知识，加到AB的已知里，再去循环，直到得到知识D
反向的话就是已知D，去条件里找想得到D需要什么条件，然后再去找想满足这个条件的前提还需要满足什么条件，直到推出需要满足条件AB
```
